

目的：
    googleplay 对apk包的容量有限制，大概50MB
    这个工具的目的就是避免手动挑选资源
    
    
    
使用方法：
    1. 注意下，这个工具是依靠 list.csv 的内容进行判断的
        暂时没有添加错误处理，一下情况可能会报错
        - 某些文件不存在
        - 无法拷贝文件（被占用之类的）
        - 创建不了目标文件夹
        - 等。。。
        
    2. 设置基本信息， 在partition.py内容的上部分
        # 目标文件夹
        COPY_DIR = "resources.googleplay"

        # 待过滤的文件夹
        SRC_DIR  = "resources.encrypt"

        # 限制大小
        # 这个是压缩后的大小
        # 应该和达到包里的大小差不多
        LMT_SIZE = 13 * 1024 * 1024

    3. 设置必须保留的文件们，通过正则表达式过滤，在函数 createFilter
    ** 被这些表达式匹配的文件们 '''一定会''' 拷贝到最终生成的文件夹里 -> COPY_DIR

        filter.addregex(r"^ccbi/.*$")     -> 保留 ccbi 文件夹下的所有文件
        filter.addregex(r"^ccbResources/login_ui/.*$")      -> 保留 ccbResources/login_ui 文件夹下的所有文件
        filter.addregex(r"^scripts/.*$")
        filter.addregex(r"^spine/.*$")
        filter.addregex(r"^table/.*$")
        filter.addregex(r"^[\w\.]+$")    -> 保留根目录的文件

        注： 也可以保留单个文件，参见正则表达式
